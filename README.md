# 稚影 (Zhiying) - 成神经细胞瘤多模态辅助诊断平台

稚影是一款结合了病理图像（WSI）与临床指标的多模态 AI 辅助诊断工作站。

## 🚀 核心亮点
- **双端适配**：包含 PC 端专业诊断站与 Mobile 端智能助手（对话流交互）。
- **可解释性 AI**：展示逻辑推导过程（Chain of Thought），而非单纯的预测结果。
- **商业级体验**：内置登录系统、历史记录溯源、附件模拟上传等功能。

## 🛠️ 如何运行
1. **克隆项目**: `git clone [你的仓库链接]`
2. **安装依赖**: `npm install`
3. **启动项目**: `npm run dev`

## 📱 移动端预览
在浏览器打开后，按 `F12` 进入移动端仿真模式，并**重新刷新页面**即可切换至“稚小影”对话界面。
