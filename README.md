# 稚影 (Zhiying) - 成神经细胞瘤多模态辅助诊断平台 🔬

稚影（Zhiying）是一款专为病理科医师设计的智能辅助诊断系统。本项目不仅实现了高分辨率 WSI 图像的展示，更核心地引入了 **可解释性 AI 推理链 (Chain of Thought)**，实现了临床文本与病理图像的多模态联合推断。

> **当前版本**：v1.2.5 (引入 Vue Router 跨端隔离架构)

## 🌟 核心特性

- **多模态分析**：结合患儿年龄、LDH/NSE 生化指标与 WSI 病理图像进行综合风险评估。
- **可解释性推理 (CoT)**：实时展示 AI 的逻辑推导过程，而非仅仅给出一个冷冰冰的概率分数。
- **零耦合跨端架构**：
  - **PC 端**：专业医生工作站，双栏布局，沉浸式分析。
  - **移动端 (Mobile)**：模拟商业智能助手，采用对话流交互，支持登录拦截与历史记录溯源。
  - **动态引导页 (Portal)**：科技感入口，支持一键分发流量至不同终端界面，互不干扰。

## 🚀 快速开始 (运行指南)

请确保你的电脑上已经安装了 [Node.js](https://nodejs.org/) (建议版本 v16.0 及以上)。

### 1. 克隆项目
将项目下载到你的本地环境：
```bash
git clone git@github.com:ZDH520a/zhiying-frontend.git
2. 进入项目目录
Bash
cd zhiying-frontend
3. 安装依赖包 (关键步骤)
项目包含 Vue Router、Tailwind 等必要库，运行前必须先安装依赖：

Bash
npm install
4. 启动本地开发服务器
Bash
npm run dev
运行成功后，终端会给出一个本地访问地址（通常是 http://localhost:5173/）。

📱 界面导航与预览说明
本项目采用 Vue Router 进行多端独立管理。启动项目后，请通过浏览器访问以下路径：

🏠 1. 系统引导页 (Portal)
访问地址: http://localhost:5173/

说明: 默认入口。极具科技感的导航中心，在此点击卡片可选择进入 PC 工作站或移动端助手。

💻 2. PC 专业工作站
直接访问: http://localhost:5173/pc

说明: 建议在全屏模式下预览，以获得最佳的双屏病理阅片与数据面板体验。

📱 3. Mobile 移动终端
直接访问: http://localhost:5173/mobile

⚠️ 预览必读操作:

进入该页面后，请按下键盘的 F12 打开开发者工具。

点击左上角的 “切换设备仿真” 图标（快捷键 Ctrl+Shift+M），选择一款手机（如 iPhone 14 Pro）。

必须按 F5 刷新页面，体验完整的移动端丝滑滚动与对话流交互体验！

🛠️ 技术栈清单
前端框架: Vue 3 (Composition API)

路由管理: Vue Router 4

构建工具: Vite

CSS 引擎: Tailwind CSS
